Learn about the LiDAR Data Service

The LiDAR Data Service provides notebooks for working with FDRI LiDAR datasets — point clouds and derived surface models such as terrain and canopy height — with links to the relevant GitHub repositories.

Use the notebooks to render and interrogate surfaces, or clone a repository to integrate LiDAR products into your own catchment analysis.

An annotated walkthrough for this tool is on the way. In the meantime, open the tool to explore, or try a Data Story.

FDRI Data Stories

FDRI Data Stories are step-by-step demonstrations of the infrastructure, drawn from real case studies of hydrology and data science in action. They are a good way to see how the tools work in practice.
