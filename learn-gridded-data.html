Learn about the Gridded Data Service

The Gridded Data Service provides notebooks for visualising FDRI gridded datasets — such as gridded rainfall and potential evapotranspiration — with links to the relevant GitHub repositories.

Open a notebook to explore the data interactively in your browser, or clone the repository to run the same analysis in your own environment with your own parameters.

An annotated walkthrough for this tool is on the way. In the meantime, open the tool to explore, or try a Data Story.

FDRI Data Stories

FDRI Data Stories are step-by-step demonstrations of the infrastructure, drawn from real case studies of hydrology and data science in action. They are a good way to see how the tools work in practice.
