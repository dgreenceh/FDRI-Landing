Access and analyse whole-system hydrological data from across the UK

This platform provides a range of tools and resources to support access to UK-wide hydrological datasets and analysis across a range of data types, including time series, geospatial, gridded data, and LiDAR.

Key features include: a Searchable Catalogue with filtering tools to help you find and understand diverse datasets, an interactive Time Series Data Explorer with tools to help you explore historical and near real-time data, a Spatial Explorer with multi-layer data visualisation tools. Notebooks to support visualisation of Gridded and LiDAR data, with links to the relevant GitHub repositories.
