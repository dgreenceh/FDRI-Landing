Learn about the GitHub Repos

The GitHub Repos area gathers the open code behind the platform — the tools, notebooks and data services — in one place.

Clone a repository to run a tool locally, read the code to understand how a service works, or open an issue or pull request to contribute back.

An annotated walkthrough for this tool is on the way. In the meantime, open the tool to explore, or try a Data Story.

FDRI Data Stories

FDRI Data Stories are step-by-step demonstrations of the infrastructure, drawn from real case studies of hydrology and data science in action. They are a good way to see how the tools work in practice.
